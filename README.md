# Proyectos Flutter

Este repositorio contiene dos proyectos creados en Flutter como parte de la clase 1.

## ejemplo1 - Application
Proyecto creado utilizando la plantilla "Application" de Flutter.
Fue compilado en modo release y se generó su respectivo archivo APK.
La aplicación fue instalada y probada en un dispositivo móvil.

## ejemplo2 - Empty
Proyecto creado utilizando la plantilla "Empty" de Flutter.
También fue compilado en modo release generando su archivo APK.
Se verificó su funcionamiento en un dispositivo móvil.
